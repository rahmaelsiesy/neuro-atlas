📖 Reference Guide

How to Use Your Quest Board

A quick reference for when you forget how things work — no judgment.

No sections match your search.
🗺️ The Big Picture
  • This board tracks your PhD as a game. Steps → XP → Levels → PhD.
  • 7 tracks: Flagship · Methods · Network · txomics · Cursor Plan · Learning · Career. Plus custom projects.
  • 8 levels from Graduate Student (0 XP) → PhD (1400 XP). Each level = a real career milestone.
  • Your one job: complete 90-min focus blocks. The board tracks everything else.
  • Success = 2 deep work blocks per day minimum. Everything above that is a bonus.
🔁 Daily Workflow
  • Every morning: open schedule.html → see today's 3 tasks → click "▶ Start Focus" → work 90 min → log it.
  • You need 2 deep blocks for a "Solid day ✓". 3 = "Great day ★". 4 = "Beast mode 🔥".
  • Didn't finish something? Hit "Carry →" — it moves to tomorrow. No guilt, no penalty.
  • Lab days: toggle a day as "Lab Day" → your deep work cap drops to 2 so you're not overloaded.
  • Admin tasks (email, backup, docs) use a separate admin quota — they never eat your deep work blocks.
🏔️ The Quest Board
  • Each colored card = one milestone. Arrows show recommended sequence.
  • Click a card → opens the Candy Land detail path for that milestone.
  • ⚙ icon (appears on hover) → quick-edit modal: update status + add notes without leaving the board.
  • XP bar in the header fills as you complete steps. Fills completely → level up animation.
  • Progress rings on each track header show % complete for that track. Keyboard: Enter on a focused card = open it.
🍬 Candy Land Path
  • Click any milestone card to enter its step-by-step Candy Land path.
  • Your glowing avatar stands on your current step. ✓ = done. 🔒 = locked (complete steps in order).
  • Click a step → opens a drawer with the full checklist + description + "Start Focus" button.
  • Check off all checklist items → "Mark step done" button appears → earns XP.
  • Robustness Gate (red shield) at the end = Nature-publication-quality checkpoint. Must pass before the milestone reward unlocks.
🔴 Code 🟠 Figure 🟢 Wet Lab 🟡 Learning 🔵 Validation 🟣 Writing ⚫ Setup
  • Keyboard: ← → to navigate between steps.
⏱️ Focus Mode
  • Shows ONE task, a 90-min countdown, and your session count for that step.
  • Browser tab title shows the timer (⏱ 87:32 — Step Name) — visible without switching windows.
  • Session dots: ● ● ○ = 2 done, 1 remaining. Each dot = one 90-min block.
  • Timer hits 0:00 → auto-logs the session, shows your daily progress threshold, offers "Next block →".
  • Lead time banners: yellow/red warnings if a compute-heavy task (decoder, SG Booster, photobleacher) is approaching its start deadline. Keyboard: Space = start/pause timer.
📅 Weekly Schedule
  • Open every Monday morning for the weekly planning modal: set target blocks, mark lab days, assign tasks.
  • Week at a Glance: 7 columns, each showing blocks completed vs. target. Color = day quality.
  • Weekly Block Chart: bar chart of deep blocks per day over 3 weeks. This is your PhD velocity — trend up = good.
  • Daily Triage (2-min flow): pick your 3 tasks for today → done. No decision paralysis.
  • Flex buffer: 2 unassigned blocks per week absorb overflow before carry-forward triggers. Carry-forward never lets any day accumulate more than 4 blocks in the queue.
🧫 Sample Tracker
  • Tracks all samples: BD2 · DG Neurogenesis · ACC Bipolar · AD/FTD. One tab per project.
  • Each row = one sample × one brain area. Pipeline status updates as you check off steps.
  • Protocol Planner: set "complete dataset by [date]" → board calculates your protocol start date (with buffer). Never start too late.
  • Booking Planner: enter your GenePS scope booking date → board shows every key milestone date working backward (booster, photobleacher, probe hyb, antibody start…).
  • Parallelization checker: shows which protocol days are safe for 2 samples. Photobleacher (1 unit) flagged as bottleneck. Stable at 4°C for 10 days after photobleaching — board tracks this window. Google Sheets sync: paste public Sheet URL → board reads live data.

📋 Misc Tasks
  • Lives at misc.html — completely separate from the PhD board.
  • Uses your admin block quota (1–2/day) — never touches deep work blocks.
  • Three piles: "This week" (external deadline) · "Watching" (delegated) · "Someday maybe" (no deadline).
  • Delegated tasks (⬜): tracked so you can follow up, but don't count toward your day.
  • Weekly: review "Someday maybe" and ask if you can drop items entirely.

➕ Adding Projects
  • Main board → scroll to bottom → "+ Add Project" button.
  • Pre-filled templates: DG Neurogenesis and ACC Bipolar (click to use).
  • Custom: enter project name, description, emoji.
  • New project gets its own track on the board + its own tab in the sample tracker.
  • Milestone steps in custom projects are added manually via the detail page.
💾 GitHub & Saving
  • On GitHub Pages: localStorage saves all progress automatically. No account needed.
  • Daily snapshot Action: commits a JSON file every day at 6am PDT. Automatic version history.
  • Weekly review Action: creates a GitHub Issue every Monday with reflection + planning template.
  • Google Sheets sync (tracker): make sheet public (view only) → paste URL → data loads live.
  • See GITHUB_SETUP.md in the repo for step-by-step instructions. To export manually: Quest Board → "Export JSON" button → save the file.
